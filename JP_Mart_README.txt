# 🛍️ JP Mart

**JP Mart** is a student-built e-commerce website that offers a seamless shopping experience with modern UI, local storage-based cart management, and essential e-commerce features like checkout, order tracking, and profile handling. Built entirely with HTML, CSS, and JavaScript, this is a beginner-friendly and responsive project showcasing a full frontend shopping flow.

## 🚀 Features

- Product listing and shopping cart
- Checkout functionality with receipt download
- Local order history
- Profile management
- Login and signup flows
- Responsive design with clean UI
- Contact and About sections
- Designed and developed by **Jatin Patial**

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: Browser `localStorage` (No backend or database required)

## 📦 Folder Structure

📁 JP Mart/
├── index.html           # Homepage with product listings
├── script.js            # JS logic for cart, checkout, and products
├── styles.css           # All UI styling
├── login.html           # User login page
├── signup.html          # User signup page
├── profile.html         # View/edit profile
├── checkout.html        # Checkout interface and receipt generator
├── orders.html          # View past orders
├── wishlist.html        # Placeholder for future feature

## 🖥️ Installation & Usage

No installation required — just clone and open in a browser.

```bash
git clone https://github.com/jatin-patial/jp-mart.git
cd jp-mart
open index.html
```
